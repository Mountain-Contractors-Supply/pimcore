# Gotham Font Files

Place your Gotham font files here:

- `gotham-book.woff2` - Gotham Book (font-weight: 400)
- `gotham-book.woff` - Gotham Book fallback
- `gotham-medium.woff2` - Gotham Medium (font-weight: 500)
- `gotham-medium.woff` - Gotham Medium fallback

These fonts are referenced in `/public/css/style.css` via @font-face declarations.

## Supported Formats
- `.woff2` - Modern format (preferred)
- `.woff` - Fallback for older browsers

## Installation
1. Download the Gotham font files from your font provider
2. Convert to WOFF2 and WOFF formats if needed
3. Place the files in this directory
4. The CSS will automatically load them
